## Dylan's Workout Tracker
I want you to create a mobile-first workout tracker where I can enter my exercise quickly and easily and you'll log it. The logging can take place in a local file for now under the date. For each new workout, I'll start a session with the date (date picker) then I'll select: cardio or gym.
Cardio is a run or bike ride with: activity (run, bike), distance, pace.
Gym will allow me to enter new line items. Each line item is: open field for exercise name (which should fill in as I type if it's a common exercise in our library), then reps, weight, sets.
Save and it gets saved to the directory. 
The interface is super easy to manipulate with fingers at the gym. 
Use all available skills, mcp servers, and plugins to make something that looks great. 
Eventually, we'll save to a db so you can analyze over time and make suggestions for the next workout according to a plan.
